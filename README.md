# Trading_Smarter
Deep reinforcement learning framework for stock trading using Q-Learning and DQN with realistic market constraints.

This project implements reinforcement learning agents to optimize stock trading strategies using historical Tesla (TSLA) data.

Technologies:
- Python
- PyTorch
- Reinforcement Learning
- Deep Q Networks

Key Features:
- Cost-sensitive trading environment
- Reward shaping
- Exploration decay tuning

Results:
- Improved trading performance compared to baseline strategies
